Header 2: Installation
Header 3: Directly (Windows)
---
1. Install FFmpeg.
2. Install a LaTeX distribution. MiKTeX is recommended.
3. Install the remaining Python packages.
```sh
git clone 
cd manim
pip install -e .
manimgl example_scenes.py OpeningManimExample
```
