For **stable** release:  
- `python -m pip install gpt-engineer`  
For **development**:
- `git clone 
- `cd gpt-engineer`
- `poetry install`
- `poetry shell` to activate the virtual environment  
We actively support Python 3.10 - 3.12. The last version to support Python 3.8 - 3.9 was 0.2.6.
